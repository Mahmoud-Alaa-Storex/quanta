Installing Quanta

Installing Quanta is a breeze, and it will take you no more than 15 minutes!

Requirements

- Apache SOLR 4.x
- PHP 5.6+ (advice: use 7!)

Installation

1) clone the latest version of the software from https://github.com/gattoplano/quanta into your apache www folder.

2) inside the /quanta folder, create a sites folder (or link it to any existing folder in your system using a symlink). This will contain all your sites.

3) chmod 

4) Done! Quanta is installed. Incredible, but true!
