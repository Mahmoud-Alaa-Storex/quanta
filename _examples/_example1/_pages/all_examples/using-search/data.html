Using Search

The [SEARCH] tag allows you to run a search in the site:

[SEARCH:search-results]

You have to specify a node where to display search results, i.e.:

[SEARCH:search-results] will load the results in the [LINK:search-results] node.

In order to display results, you need to use the [SEARCHRESULTS] tag.

The output will then look like this:

--------

[SEARCHRESULTS]
