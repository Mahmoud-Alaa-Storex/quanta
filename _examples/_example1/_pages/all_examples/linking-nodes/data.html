Linking nodes and external content

To link a node to another you should use the [LINK|showtag] tag.

This is the output of [LINK|showtag:home]

[LINK:home]

The link text is automatically calculated from the target node. But it can also be forced to something else, using the |title attribute.

This is the output of [LINK|title=Go back to the homepage|showtag:home]

[LINK|title=Go back to the homepage:home]

You can also use external URLs as an argument.

This is the output of [LINK|title=Go to the QuantaCMS home page|showtag:http://www.quantacms.org]

[LINK|title=Go to the QuantaCMS home page:http://www.quantacms.org]
