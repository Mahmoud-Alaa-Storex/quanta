Counting children of a Node

You can count how many children a node has by using the [COUNT] tag.

In example, by running this command on home: [COUNT:home]

We get this result: [COUNT:home]

Reflecting the number of elements in the whole [LINK|title=TREE:lists-and-trees] of the [LINK|title=home:home] node:

[TREE|nolinks|box=100xauto|border=1|padding=20|centered:home]

It's also possible to setup mindepth or maxdepth parameters, to choose at which level to start counting.

[COUNT|maxdepth=1:home] will return [COUNT|maxdepth=1:home]

and

[COUNT|mindepth=2:home] will return [COUNT|mindepth=2:home]
